# task management project
# spec

Taskmangement is a website deployed by leni.menage@outlook.fr. it's supposed to help the users to organize and list their tasks or their shopping list.
the website is divised in three part.
  - the main section: its the main part of the website used to write tasks. it's composed of two elements, the patterns and the tasks.
    patterns are the tasks groups and tasks are the items wich people need
  - The pattern section: can add and saves patterns
  - Tab section: at the top of the web page there is the tab which can change the pattern to have two lists different at the same time
